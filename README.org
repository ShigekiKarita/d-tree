* Decision tree library for D

getting started

#+begin_src
$ curl https://dlang.org/install.sh | bash -s # install dlang compiler
$ source ~/dlang/<your-compiler>/activate
$ cd example/plot_dtree_boundary
$ dub run
$ xdg-open *.png
#+end_src
